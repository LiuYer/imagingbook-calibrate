This Java library implements the geometric part of Zhang's camera calibration algorithm, described in Z. Zhang, "A flexible new technique for camera calibration", IEEE Transactions on Pattern Analysis and Machine Intelligence, 22(11), pp. 1330-1334, 2000. The implementation does not include interest point detection and point correspondence calculations. See this extensive report for details.

Dependencies:

Depends on ImageJ, the ImagingBook Common library and Apache Commons Math.

Installation:

Place imagingbook-common.jar and imagingbook-calibrate.jar in ImageJ/jars/.
Place the sample plugins in ImageJ/plugins/ and restart ImageJ.
